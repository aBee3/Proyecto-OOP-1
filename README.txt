README

Mi proyecto es un programa que traduce tu texto a diferentes formas de texto, usando alfabetos de otros idiomas y otros animales (quizás).
Por ejemplo, tu puedes dar la palabra "CHOCOLATE" y traducirlo al abecedario japonés, chino y koreano de forma literal.
Estarán las clases:
1. Poema Asiático
1.1 chino
1.2 japonés
1.3 coreano

2. Poema animal
2.1 gato
2.2 perro

Donde la clase asiático y animal son clases abstractas. Cada respuesta o nueva palabra se calcula de forma diferente para cada abecedario y regresa textos diferentes.
Un libro está compuesto de poemas en diferentes abecedarios, puedes reescribir cada poema con diferente abecedario, puedes calcular en que abecedario está más completo.
